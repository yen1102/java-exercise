貓貓桌寵規則：

初始設定：
剛出生初始金幣為100
體力值、飽食度皆為100%

關於飽食度、體力值：
貓貓隨時都在消耗飽食度
體力值在睡覺時會補充能量，其餘時間也會消耗體力值
食物部分有三種，分別為：
貓貓飼料：飽食度+50%
貓貓罐頭：飽食度+25%、體力值+25%
貓條：體力值+50%
每種食物都會花費50金幣
請注意：當貓貓在睡覺、吃飯時你去打擾他（左鍵點他或拖移他），他便會停止睡覺、吃飯

關於探險：
每次探險會消耗30%體力以及20%飽食度
探險時間為5分鐘
其中間可以退出探險模式，但是不會返還體力值及飽食度（請三思）
探險結束後可獲得10~210金幣

關於生病：
貓貓沒有食慾吃東西，但因為不動的關係飽食度下降速度變慢
休息養身體，體力值緩慢增加
生病時沒力氣去探險，不管用什麼方式（除了病好）都沒辦法叫醒貓貓
生病時會多出兩種選項：
1.看醫生，不會立刻好，但是會減短生病時長（花費300金幣）
2.貓病特效藥，立刻見效，馬上就好（花費1000金幣）
不管他也會好，但是會比較久
